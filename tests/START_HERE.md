## Success Looks Like

### If All Metrics Pass:

```
✓ Clarity ≥70%
✓ Autonomy ≥70%      ← CRITICAL for human agency
✓ Effectiveness ≥70%
✓ Efficiency ≤2 cycles
✓ Dignity PASS 100%  ← NON-NEGOTIABLE
```

**Decision: DEPLOY Reset Diplomatique to production**

### If 1-2 Metrics Fail:

Identify which ones. Review session notes for root cause. Make targeted fixes. Re-test 2-3 users.

**Decision: ITERATE + re-test**

### If 3+ Metrics Fail:

Too many issues. Major redesign needed.

**Decision: REDESIGN + restart testing**

## FAQ (5 Minute Answers)

**Q: Why 7 users?**
A: MVP is 3. Seven gives confidence. More is diminishing returns.

**Q: How long does each session take?**
A: 90 min planned, usually 60-75 min actual. Plan 2 hours with setup.

**Q: What if results are bad?**
A: That's why we test! Better to find issues now than after deployment.

**Q: Can we start smaller?**
A: Yes, 3 users is MVP. Then scale to 7 for confidence.

**Q: What happens after?**
A: If all metrics pass → Deploy. If 1-2 fail → Iterate. If 3+ fail → Redesign.

**Q: Can we run sessions in parallel?**
A: Only with multiple facilitators. Sequential is better (you learn each session).
