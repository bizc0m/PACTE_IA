### Decision Logic
```
If ALL 5 core metrics pass (Clarity, Autonomy, Effectiveness, Efficiency, Dignity):
  → DEPLOY Reset Diplomatique
  → Document success for audit
  → Proceed to production

If 1-2 metrics fail:
  → Identify which metrics failed
  → Review session notes for root causes
  → Propose targeted fixes
  → Re-test with 2-3 users (specific focus)
  → Re-check metrics

If 3+ metrics fail:
  → Major redesign needed
  → Revert to theory phase
  → Design new approaches
  → Plan new testing cycle
```

## Contingency Plans

### If Participant Cancels
- [ ] Contact backup candidate immediately
- [ ] Reschedule within 24 hours
- [ ] Confirm backup is ready to go

### If Equipment Fails
- [ ] Switch to backup (phone audio)
- [ ] Reschedule video portion if critical
- [ ] Continue session with audio notes

### If Participant Gets Frustrated
- [ ] STOP immediately
- [ ] Ask if they want to continue
- [ ] Document exactly what happened
- [ ] This is valuable data, not a failure

### If Reset Scores Very Low (After 2 Sessions)
- [ ] Meet with team to analyze

- [ ] Continue testing with modifications noted
