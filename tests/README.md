## Success Criteria

Reset Diplomatique is **PRODUCTION READY** if:

```
✅ Clarity ≥70% (on ≥70% of sessions)
✅ Autonomy ≥70% (on ≥70% of sessions)      ← CRITICAL for PACTE_IA
✅ Effectiveness ≥70% (on ≥70% of sessions)
✅ Efficiency ≤2 cycles (on ≥80% of sessions)
✅ Dignity PASS (on 100% of sessions)        ← NON-NEGOTIABLE
```

**All 5 checks must pass.** If 1-2 fail: iterate and re-test. If 3+: redesign.

## The 6 Metrics Explained

### 1. Clarity Score (0-100)
**Does user understand the diagnosis?**
- Threshold: ≥70%
- Questions: "Did you understand what the IA identified?"
- Why: Without clarity, Reset fails at the first step

### 2. Autonomy Score (0-100)
**Can user actually choose between options?**
- Threshold: ≥70%
- Questions: "Did you feel like you had real choice?"
- Why: Core to PACTE_IA (human agency)

### 3. Effectiveness Score (0-100)
**Did Reset actually solve the problem?**
- Threshold: ≥70%
- Questions: "Did it resolve your frustration?"
- Why: Main objective of Reset

### 4. Efficiency Score (0-100)
**How many cycles did it take?**
- Threshold: ≤2 cycles (= Efficiency ≥75%)
- Measure: Count cycles, track time
- Why: Practical usability

### 5. Trust Score (0-100)
**Would user use IA again?**
- Threshold: ≥70% (lower threshold, bonus metric)
- Questions: "Would you recommend this?"
- Why: Long-term adoption signal

### 6. Dignity Score (Pass/Fail)
**Did user feel respected?**
- Threshold: PASS on 100% (non-negotiable)
- Violations: Apology script, manipulation, coercion, gaslighting
- Why: Ethical requirement, PACTE_IA core

## FAQ

**Q: Do we really need 7 users?**
A: MVP is 3. Seven gives confidence. More is diminishing returns. Seven is sweet spot.

**Q: How long does each session really take?**
A: 90 min planned, usually 60-75 min actual. Budget 2 hours with setup/teardown.

**Q: What if Reset doesn't score well?**
A: That's why we test! Better to find issues before launch.

**Q: Can we run sessions in parallel?**
A: Only if you have 2+ facilitators. Sequential is better (you learn after each).

**Q: What happens to the data?**
A: Stored securely, anonymized in final report. Participants can request deletion.

**Q: Who decides if we deploy after testing?**
A: Executive sponsor + test results. All 5 success criteria must pass.
